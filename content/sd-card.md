# SD Card Controller Solution

the **aio** SD memory card controller fully supports the latest SD 6.1 specifications, and also supports full coverage of card capacity including High Density SD Memory Card (SDHC) and Extended Capacity SD Memory Card (SDXC).

It supports Toggle DDR 2.0 to achieve high performance. Its powerful LDPC engine improves endurance of NAND flash. And the efficient power saving modes reduces system power consumption during idle time. It is suitable for developing high-performance, low-power, and high-reliable SD memory card.

![SD Card Diagram](/images/products/sd1.jpg)

---

## Key Features

- **Supports SD6.1 Interface**
  - Application Performance Class (A2)
  - Random IOPS Read/Write >4K/2K
  - Low Voltage Signal (LVS)
- **Supports Bus Speed Mode:** Default Speed mode, High Speed mode, SDR12, SDR25, SDR50, SDR104, DDR50, DDR200
- **Advanced LDPC ECC engine**
- **Supports the Toggle DDR 2.0 interface specification**
- **Supports single channel 8-bit data bus**
