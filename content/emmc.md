# eMMC Controller Solution

the **aio** eMMC controller is a high-performance eMMC controller designed for embedded NAND flash storage applications. It fully supports eMMC 5.1 version of JEDEC standard with newly introduced features such as Command Queuing (CQ), Enhanced Strobe and so on.

the aio eMMC controller supports various flash interfaces such as ONFI 3.0 (NV-DDR1, NV-DDR2) and Toggle DDR 2.0 and its ECC engine enables efficient flash management and improves endurance of NAND flash. Complete & enhanced vendor command set enables customer’s test modes, flexibility for failure analysis and system development much easier in embedded configuration.

## Key Features

- **Performance**: High IOPS via Command Queuing (CQ)
- **Compatibility**: Supports eMMC5.1 features fully compatible with eMMC4.5/5.0
- **Speed**: Supports HS400 mode with Dual Data Rate (DDR, up to 200MHz clock speed)
- **Interface**: Supports the ONFI 3.0 (NV-DDR, NV-DDR2) & toggle DDR 2.0 interface specification
- **Security**: Write protection for security
- **Efficiency**: Supports Power saving mode and single-channel 8-bit data bus
