## Product Overview
The AIO eMMC controller is a high performance eMMC controller designed for embedded NAND flash storage applications. It fully supports eMMC 5.1 version of JEDEC standard with newly introduced features such as Command Queuing (CQ), Enhanced Strobe and so on. And it has featured flash management FTL (Flash Transition Layer) for embedded application and enable manages high reliability with noticeable Wear Leveling, Bad Block Management and strong ECC. Again, it achieves super high performance, especially random IOPS.

The AIO eMMC controller supports various flash interfaces such as ONFI 3.0 (NV-DDR1, NV-DDR2) and Toggle DDR 2.0 and its ECC engine enables efficient flash management and improves endurance of NAND flash. Complete & enhanced vendor command set enables customer’s test modes, flexibility for failure analysis and system development much easier in embedded configuration.

## Features
- Increasing Write Speed through SLC Buffering
- Supports eMMC5.1 features fully compatible with eMMC4.5/5.0
- Supports HS400 mode with Dual Data Rate (DDR, up to 200MHz clock speed)
- Supports the ONFI 3.0 (NV-DDR, NV-DDR2) & toggle DDR 2.0 interface specification
- Advanced embedded ECC engine
- Supports Power saving mode
- Supports wide vendor command sets
- Supports single channel 8-bit data bus
- Write protection for security

## Temperature
- Operation: -25℃ ~ 85℃
- Storage without operation: -40℃ ~ 85℃
